### 项目描述

基于 LangGraph + FastAPI 构建的全自动 24 小时不间断 AI 新闻播客系统，集成 Chroma 向量数据库与 SSE 流式 AI 对话助手。通过四阶段 Agent 流水线、TTS 适配器模式、双存储架构（SQLite + Chroma 日库/总库）、HTTP Live Streaming 与 Chat RAG 智能问答联动，实现新闻自动采集 → 双人对话脚本（全中文播报）→ 多 Provider 语音合成 → 永不间断流式播放 + SSE 流式新闻知识库智能问答的完整业务闭环。纯 Python 技术栈，支持模型服务热切换，适配器模式兼容多家 TTS 引擎。

### 简历亮点关键词

LangGraph Agent 流水线, StateGraph 子图编排, TypedDict 类型安全状态管理, 四阶段 Pipeline（采集/脚本/TTS/播单）, 两阶段 Prompt 工程（大纲→对白）, 双人对话脚本生成（主持人+技术专家）, Fallback 降级模板（4 档中英文）, 全中文播报策略, TTS 适配器模式（Abstract Adapter + Registry）, Edge TTS / 豆包 / OpenAI TTS 可插拔, 多角色独立语音分配, Chroma 向量数据库双 Collection 架构（日库+总库）, SSE 流式 AI 问答（text/event-stream）, RAG 语义检索 + LLM token 级流式输出, HTTP Live Streaming 24/7 不间断播放, FIFO 智能队列（最新优先）, 启动自动化检测时间窗口, 数据归档系统（is_archived 逻辑删除 + Cron 自动归档）, 新闻三级去重（Hash + SequenceMatcher 0.85 + DB）, LLM 精排 + 关键词预排, 模型服务热切换（运行时动态解析）, aiosqlite 异步 SQL 无 ORM, pydub + ffmpeg 音频拼接（300ms 交叉淡入淡出, -16dBFS 归一化）, APScheduler 定时调度 + Cron 归档, FastAPI + StreamingResponse + SSE, 原生 JS SPA 5 Tab 管理面板, ruff + pytest 16 用例 100% 通过
